# JSON Format - 简单强大的 JSON 查看器

[English](./README.md)

一款轻量级、高性能且功能丰富的 Chrome JSON 格式化扩展，完美适配 Manifest V3。

该扩展可以自动检测浏览器中的原始 JSON 内容，并将其转换为美观、易读的可折叠树形结构。帮助开发者更高效地查看和调试 JSON 数据。

## 核心功能

- **自动格式化**：自动将 JSON 响应转换为可读的树形结构。
- **JSON 编辑器**：专门的编辑器页面 (`editor.html`)，支持手动输入 JSON 进行格式化。
- **快捷键支持**：在任何页面按下 `Alt + J` 即可快速打开 JSON 编辑器。
- **可折叠树形视图**：轻松展开和折叠 JSON 对象与数组。点击 `-`/`+` 图标或 `{...}` 占位符即可切换。
- **语法高亮**：为键、字符串、数字、布尔值和 null 值提供语法高亮。
- **浮动工具栏**：右上角提供快捷操作：
  - **Parsed/Raw 切换**：在格式化树和原始文本之间快速切换。
  - **全部展开/折叠**：一键控制所有节点的显示状态。
  - **复制**：一键复制格式化后或原始的 JSON 内容到剪贴板。
- **历史记录**：自动保存最近查看的 JSON。从侧边栏即可快速访问。
- **Diff 对比**：支持将当前 JSON 与历史记录并排对比。
  - **同步滚动**：左右两边内容同步滚动，方便行对行比对。
  - **对齐视图**：键值对齐显示，方便对比。
  - **颜色标记**：新增（绿色）、移除（红色）和修改（黄色）字段一目了然。
  - **同步折叠**：在对比视图中折叠节点时，保持结构对齐。
- **链接自动识别**：自动识别字符串中的 URL 并转换为可点击链接。
- **历史记录管理**：
  - **可收起面板**：支持将右侧面板彻底收起，节省空间。
  - 查看每条记录的时间戳和来源 URL。
  - 支持拖拽调整面板宽度。
  - 一键清空历史记录。

## 安装

### 通过 Chrome 应用商店安装
[Chrome 应用商店下载](https://chromewebstore.google.com/detail/json-format/cacimhdphkcihjfpnpmmndgjjnnfoobm)

### 手动安装 (开发者模式)

1. 克隆或下载本项目代码。
2. 打开 Chrome 并访问 `chrome://extensions/`。
3. 开启右上角的 **开发者模式 (Developer mode)**。
4. 点击 **加载已解压的扩展程序 (Load unpacked)**。
5. 选择项目目录 (`JsonFormat`)。

## 使用说明

1. 打开任何返回原始 JSON 响应的 URL（例如 API 接口）。
2. 扩展程序会自动格式化 JSON 内容。
3. 使用 **右侧面板** 查看历史记录。
4. 点击历史记录项进入 **Diff 对比模式**。
5. 拖拽历史面板边缘可调整宽度。

## 构建打包

如需为 Chrome 应用商店打包项目（压缩 JS/CSS 并生成 ZIP）：

```bash
npm install
node build.js
```

生成的 `JsonFormat.zip` 文件将位于项目根目录。
