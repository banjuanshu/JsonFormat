# JSON Format - 简单强大的 JSON 查看器

[English](./README.md)

一款轻量级、高性能且功能丰富的 Chrome JSON 格式化扩展，完美适配 Manifest V3。

该扩展可以自动检测浏览器中的原始 JSON 内容，并将其转换为美观、易读的可折叠树形结构。帮助开发者更高效地查看和调试 JSON 数据。

## 核心功能

- **自动格式化**：自动将 JSON 响应转换为可读的树形结构。
- **折叠树形视图**：轻松展开和折叠 JSON 对象与数组。点击 `-`/`+` 图标即可切换显示。
- **语法高亮**：为键、字符串、数字、布尔值和 null 值提供不同颜色的高亮显示。
- **历史记录**：自动保存您最近查看的 JSON。从侧边栏即可快速访问历史数据。
- **Diff 对比**：支持将当前 JSON 与任意历史记录进行并排对比。
  - **对齐视图**：键值对齐显示，方便对比。
  - **颜色标记**：新增（绿色）、移除（红色）和修改（黄色）字段一目了然。
  - **同步折叠**：在对比视图中折叠节点时，保持结构对齐。
- **链接自动识别**：自动识别字符串中的 URL 并转换为可点击链接（在新标签页打开）。
- **历史记录管理**：
  - 查看每条记录的时间戳和来源 URL。
  - 支持拖拽调整历史面板宽度。
  - 一键清空历史记录。

## 安装

### 通过 Chrome 应用商店安装
[Chrome 应用商店下载](https://chromewebstore.google.com/detail/json-format/cacimhdphkcihjfpnpmmndgjjnnfoobm)

### 手动安装 (开发者模式)

1. 克隆或下载本项目代码。
2. 打开 Chrome 并访问 `chrome://extensions/`。
3. 开启右上角的 **开发者模式 (Developer mode)**。
4. 点击 **加载已解压的扩展程序 (Load unpacked)**。
5. 选择项目目录 (`JsonFormat`)。

## 使用说明

1. 打开任何返回原始 JSON 响应的 URL（例如 API 接口）。
2. 扩展程序会自动格式化 JSON 内容。
3. 使用 **右侧面板** 查看历史记录。
4. 点击历史记录项进入 **Diff 对比模式**。
5. 拖拽历史面板边缘可调整宽度。

## 构建打包

如需为 Chrome 应用商店打包项目（压缩 JS/CSS 并生成 ZIP）：

```bash
npm install
node build.js
```

生成的 `JsonFormat.zip` 文件将位于项目根目录。
